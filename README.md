## Traductor de Lengua de Señas a Voz

Esta es una aplicación de demostración que utiliza un modelo de Machine Learning para traducir imágenes del lenguaje de señas (letras individuales) a texto, y luego convierte ese texto a voz.

**Tecnologías utilizadas:**
*   Python, Flask, Gunicorn
*   TensorFlow/Keras
*   OpenCV, MediaPipe
*   Spacy, gTTS
*   Docker
